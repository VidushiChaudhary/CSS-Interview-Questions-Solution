Q1. Difference between flex row and column.
Ans. Flex-Direction:- The flex-direction property specifies the direction of the flexible items.
CSS Syntax:- flex-direction: row|row-reverse|column|column-reverse|initial|inherit;

The flex-direction Property

Flex-row: Default value. The flexible items are displayed horizontally, as a row.

Flex-column: The flexible items are displayed vertically, as a column.

Q2. Explain inline , internal and external stylesheet


Inline CSS:- Applied directly to a specific HTML element using the style attribute. Inline CSS is best for quick styling tweaks and debugging.

Internal CSS:- Specified in the head section of an HTML document using a style element. Internal CSS is best for single-page projects, prototypes, and overriding external styles.

External CSS:-Stored in a separate .css file and linked to multiple HTML documents using a link element. External CSS is best for multi-page websites, large projects, and maintainability.

Q4.Difference between absolute and relative positioning.
Relative positioning is a CSS technique that allows an element to be adjusted from its normal position. The syntax for relative positioning is position: relative;

Absolute positioning is another CSS technique that adjusts an elements position relative to its parent. If no parent element is present, the document body is used as the parent.

Q6. What is the difference between padding and margin?
The main difference between padding and margin is the placement of the space they create:
Padding: The space between the content and the border of an element.
Margin: The space around an element, between the border and outer components.
